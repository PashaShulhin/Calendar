 Calendar App
This is an interactive calendar application built with React. Users can view daily events, switch between days, and see a detailed schedule interface.

⚙️ Technologies Used
React

Vite

JavaScript (ES6+)

CSS

React Router

REST API 

🚀 How to Run Locally
bash
Copy
Edit
npm install
npm run dev




📁 Project Structure
src/pages/DayView/DayView.jsx — Main day view page

vite.config.js — Configuration file

src/assets/ — Icons and images
